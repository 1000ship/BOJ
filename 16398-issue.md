# 16398 행성 연결



### 두 가지 최소스패닝트리 알고리즘

* Kruskal 시간 복잡도 **O(E log₂E)**

  -> *간선 숫자*에 영향

* Prim 시간 복잡도 **O(V^2)**, **O(E log₂V)**

  -> *정점 숫자*에 영향



### 문제 원인

- 정점의 갯수 1,000개

- 간선의 갯수 1,000 + 999 + ... + 2 + 1 = 500,500개

- 크루스칼 사용 시, 반복횟수 약 500,500 * 18 = 1천만회

  -> Pypy로 정답처리된게 신기한 셈

- 프림 사용 시, 반복횟수 약 1백만회



### 결론

: 최소스패닝트리 문제 풀이 시, 간선의 갯수와 정점의 갯수를 고려하여 알고리즘을 선별해 사용하자
